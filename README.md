# Throw The Darts
Game ini merupakan sebuah game yang dibuat untuk memenuhi _final project_ dari mata kuliah Pemrograman Lanjut Semester 2 di Teknik Komputer ITS. Dirancang mirip dengan game Knife Hit yang ramai dimainkan pada masanya, game ini menggunakan Ubuntu (WSL) dan dimainkan dengan cara memencet tombol spasi pada keyboard.
